# 🛍️ Essence Market - Full Stack E-commerce

A modern full-stack e-commerce web application with product browsing, cart functionality, and backend integration.

---

## 🚀 Features

* 🛒 Product listing & details
* 🧺 Add to cart
* 🔐 User authentication
* 📦 Order management
* 📊 Admin dashboard

---

## 🛠️ Tech Stack

**Frontend:**
React.js, Tailwind CSS, Axios

**Backend:**
Node.js, Express.js

**Database:**
MongoDB / Supabase

## ⚙️ Setup

```bash
git clone https://github.com/your-username/essence-market.git
cd essence-market
```

### Frontend

```bash
cd frontend
npm install
npm start
```

### Backend

```bash
cd backend
npm install
npm run dev
```

---

## 🌐 Environment Variables

Create `.env` in backend:

```
PORT=5000
MONGO_URI=your_connection_string
JWT_SECRET=your_secret
```
